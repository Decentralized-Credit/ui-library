# SVG Components

These SVG components all use the base SVG props defined below.

## Props Interface

```typescript
interface SVGProps {
  width?: number;
  height?: number;
  fill?: string;
  stroke?: string;
  strokeWidth?: number;
  className?: string;
}
```

## Example Usage

```tsx
import { MyIcon } from './icons/MyIcon';

// Basic usage
<MyIcon width={24} height={24} />

// With custom colors
<MyIcon fill="red" stroke="blue" strokeWidth={2} />

// With additional className to change the color
<MyIcon className="text-accent" />
```
